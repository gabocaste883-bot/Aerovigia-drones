# AEROVIGIA-DRONES Landing Page

Landing page actualizada para AEROVIGIA-DRONES · Grupo Drones y Movilidad.

## Cambios incluidos

- Se mantiene la estructura visual oscura/cian de la landing presentada al cliente.
- Se elimina la sección de licencia/certificado y el número de licencia.
- Se agrega perfil profesional de Wilmer Bello Caicedo.
- Se integran imágenes locales en `src/assets` para evitar errores de carga en Vercel.
- Se actualizan los servicios:
  1. Seguridad con drones y búsqueda en tiempo real.
  2. Protección VIP de alto perfil.
  3. Seguridad de infraestructuras.
  4. Escoltas aéreas de valores.
- Se mantiene sección de sectores, proceso, asesoría y contacto.

## Publicación en Vercel

- Framework: Vite
- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`

## Importante

Para actualizar el repositorio en GitHub, sube todos los archivos y carpetas de este paquete a la raíz del repositorio y acepta reemplazar los archivos existentes.
