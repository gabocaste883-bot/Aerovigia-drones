# AEROVIGIA-DRONES Landing Page

Landing page profesional para AEROVIGIA-DRONES · Grupo Drones y Movilidad.

## Publicación en Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`

## Imágenes incluidas

La landing incluye imágenes profesionales para los cuatro servicios:

1. Seguridad con drones y búsqueda en tiempo real
2. Protección VIP de alto perfil
3. Seguridad de infraestructuras
4. Escoltas aéreas de valores

También incluye la foto profesional de Wilmer Bello Caicedo y la imagen de la sección Cómo trabajamos.
